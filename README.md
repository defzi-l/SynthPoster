# 海报创作智能体协同系统

这是一个由多个AI智能体协同工作的海报生成系统。用户输入一段描述，系统会依次通过“拆解”、“优化”、“风格化”三个智能体进行处理，最终输出可用于图像生成的提示词。

## 工作原理
1.  **拆解智能体**：理解并结构化您的模糊描述。
2.  **优化智能体**：将结构转换为专业的英文图像提示词。
3.  **风格化智能体**：根据校园活动类型，为提示词添加合适的艺术风格后缀。

## 如何使用
在页面的文本框中输入您想要的海报描述（例如：“一场关于AI伦理的校园讲座海报”），然后点击按钮即可查看完整的协同过程和结果。
